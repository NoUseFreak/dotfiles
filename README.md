dotfiles
========

NoUseFreak's dotfiles

# What is included

## Runtime
 - handy aliasses
 - bash coloring
 - git
  - global ignore
  - autocompletion
 - history improvements
 - ssh autocomplete

## Software
 - brew
 - composer
 - hub
 - mailcatcher
 - node js
 - ssh-copy-id
 - vagrant

# Installation

```
$ git clone -b work https://github.com/NoUseFreak/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./install.sh
```
