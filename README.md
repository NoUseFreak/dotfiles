dotfiles
========

NoUseFreak's dotfiles

# What is included

## Runtime
 - handy aliasses
 - bash coloring
 - git
  - global ignore
  - autocompletion
 - history improvements
 - ssh autocomplete

## Software
 - brew
 - composer
 - hub
 - node js
 - vagrant

# Installation

```
$ git clone git@github.com:NoUseFreak/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./install.sh
```
