dotfiles
========

NoUseFreak's dotfiles

# Included

Runtime
 - handy aliasses
 - bash coloring
 - git
  - global ignore
  - autocompletion
 - history improvements

Installation
 - brew
 - composer
 - hub
 - mailcatcher
 - node js
 - ssh-copy-id
 - vagrant

# Installation

```
$ git clone git@github.com:NoUseFreak/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./install.sh
```
